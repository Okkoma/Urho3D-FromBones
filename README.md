# Urho3D Fork for FromBones Project

## Main features added
- `AnimatedSprite2D` with character mapping
- Many modifications and improvements for Urho2D
- Experimental Vulkan backend (currently does not support RenderPath)

**Urho3D** is a free lightweight, cross-platform 2D and 3D game engine implemented in C++ and released under the MIT license. Greatly inspired by OGRE and Horde3D.

Main website: [https://urho3d.io/](https://urho3d.io/)
